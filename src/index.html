ConvertSlack.to is designed for anyone who frequently copies Slack messages into platforms like Github Issues, only to find that the original formatting is lost. Simply paste your Slack message into the input box, click the button corresponding to your desired format, and your converted message will appear in the output box, ready to be copied and pasted wherever you need it. Check out our Github repository to get started or watch our demo video.
